1. Валюта проекта переименована: `Кристаллы` → `Синемы`.
   - UI: баланс, paywall, кнопки генерации.
   - Стартовый баланс нового пользователя: 25 Синем.
   - Дневной минимум при новом дне: 10 Синем.

2. Добавлена экономика списаний по AI-действиям:
   - Написать сценарий: -1 Синема.
   - Усилить текст: -1 Синема.
   - Reference prompts: -1 Синема.
   - Кастинг: -1 Синема.
   - Шаг 1: -1 Синема.

   - FB/Reels/Carousel/Stories: -1 Синема.

3. Autosave расширен:
   - сохраняет жанр, движок, формат, длительность, язык, seed, refs, frames, result state, step2Done, thumb, SEO, music и view.

   - текущие стили получили жёсткие style-lock промпты.

   - выбранный движок передаётся в Step 1 как `VISUAL ENGINE LOCK`.

   - `TIMING LOCK`: общая длительность, количество кадров, около 3 сек/кадр, VO должен помещаться в тайминг.

7. Reference Studio усилен:
   - добавлен блок `Reference prompts до загрузки`.
   - можно получить prompts для персонажей/локации/стиля до загрузки референсов.
   - добавлен Multi Character References: 2–6 фото персонажей.
   - анализ референсов теперь создаёт несколько DNA-lock, а не только один.

8. Навигация после Шага 1:
   - добавлена кнопка `Вперёд: Шаг 2 / PRO-промпты`.
